You are working inside an existing repository that currently contains a **static HTML club website**. Your task is to **convert the entire project into a modern full-stack application using Next.js, Supabase, and Vercel**, while preserving the existing UI where possible.

Follow the requirements below carefully.

---

# Project Goal

Transform the static HTML club website into a **Next.js full-stack application** with:

* Next.js (App Router)
* Supabase (database + auth)
* Vercel deployment
* Dynamic event calendar
* Admin dashboard for executive members
* Navigation system

The result should be production-ready and deployable to Vercel.

---

# Tech Stack Requirements

Frontend:

* Next.js 14+ (App Router)
* React Server Components where appropriate
* Tailwind CSS for styling
* FullCalendar or a similar React calendar library

Backend:

* Supabase

  * PostgreSQL database
  * Authentication
  * Row Level Security (RLS)

Deployment:

* Vercel

Package manager:

* npm
